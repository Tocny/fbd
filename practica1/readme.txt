Descripción del poryecto:
Un programa que lee un arhivo con al menos 10 registros de vuelos, donde solo se
muestra los vuelos con origen de Ciudad de México y la duración promedio de todos
los vuelos registrados en el archivo.

Para compilar:
    javac -d bin $(find src -name "*.java")
Para ejecutar:
     java -cp bin main.java.Main
